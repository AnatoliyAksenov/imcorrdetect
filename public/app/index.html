RID story

We analyzed if an image had been processed using Gaussian blur. At stage 1, we used dataset CIFAR-10 and split every image into 32*32 pixel squares. Then we used a convolutional neural network to apply weights to every square, marking non-retouched layers of the squares as '0' and retouched layers as '1', which is the mod_vgg19 procedure. However, it did not produce the predicted results.


At stage 2 we used another handmade dataset and split images in this set into 64*64 pixel squares. Then we again applied weights to every square.
